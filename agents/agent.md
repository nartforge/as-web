# Assyte Dev Website Agent Plan

## Amaç
Assyte Dev için premium kalitede, sade, hızlı, modern ve profesyonel bir web sitesi üretmek.

## Genel Tasarım Kuralları
- Ana tema beyaz, turkuaz ve mavi paletinde olmalı.
- AMOLED dark mode gerçek siyah arka plan kullanmalı.
- Tasarım temiz, güven veren ve premium hissiyatlı olmalı.
- Sayfalar mobil uyumlu olmalı.
- Her sayfada Discord yönlendirmesi net görünmeli.

## Sayfalar
1. Ana Sayfa
   - İlk ekranda Discord CTA göster.
   - Discord linki: https://discord.gg/assytedev
   - Metin: “duyurular , ürün güncellemeleri ve destek için discord sunucumuza katılın!”
   - Assyte Dev nedir alanı ekle.
   - Güncelleme alan ürünler bölümü ekle.
   - Duyurular bölümü ekle.
   - Scroll reveal animasyonları kullan.

2. Mağaza
   - Ürün kartları, fiyat, etiket ve satın alma butonları.
   - Satın alma şimdilik Discord yönlendirmesi yapabilir.

3. Ürünler
   - Minecraft sistemleri, Discord botları, web arayüzleri, grafik paketleri.
   - İleride filtrelenebilir kategori yapısına uygun tasarla.

4. Wiki
   - Kurulum, yapılandırma ve destek rehberleri.
   - Dokümantasyon genişletmeye uygun olmalı.

5. Destek
   - Discord odaklı destek çağrısı.
   - Kullanıcıyı hızlıca destek kanalına yönlendir.

## Teknik Kurallar
- Statik HTML/CSS/JS kullanılmalı.
- Framework zorunlu değil; sade ve hızlı kalmalı.
- Kodlar okunabilir, düzenli ve klasör yapısı temiz olmalı.
- Tema seçimi localStorage ile kaydedilmeli.
- Harici bağımlılığa ihtiyaç duyulmamalı.

## Geliştirme Sonrası Kontrol
- Tüm linkler çalışıyor mu kontrol et.
- Responsive görünümü test et.
- Dark mode geçişini test et.
- Sayfa başlıklarını ve aktif menüleri kontrol et.
- Zip paketinde README ve agent dosyası bulunduğundan emin ol.
